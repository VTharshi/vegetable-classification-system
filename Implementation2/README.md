# Implementation 2 - Vegetable Classification System

## Overview
This implementation focuses on classifying vegetable images using a Convolutional Neural Network (CNN). The model is trained on a vegetable image dataset and is capable of predicting the correct vegetable category from an input image.

## Objectives
- Build a CNN-based image classification model.
- Train the model using vegetable image datasets.
- Evaluate model performance using validation and testing data.
- Predict vegetable classes from unseen images.

## Technologies Used
- Python
- Jupyter Notebook
- TensorFlow
- Keras
- NumPy
- Matplotlib
- OpenCV

## Files Included
- `mini_project_implementation2.ipynb` – Main notebook containing data preprocessing, model training, evaluation, and prediction.

## Dataset Preparation
1. Collect vegetable images.
2. Organize images into training, validation, and testing folders.
3. Resize and preprocess images before training.

## Model Architecture
The implementation uses a Convolutional Neural Network (CNN) consisting of:
- Convolution Layers
- Max Pooling Layers
- Dense Layers
- Softmax Output Layer

## Performance Evaluation
The model performance is evaluated using:
- Training Accuracy
- Validation Accuracy
- Loss Curves
- Classification Metrics

## How to Run
1. Open the notebook in Jupyter Notebook or Google Colab.
2. Install the required libraries.
3. Run all cells sequentially.
4. View training results and predictions.

## Results
The model successfully learns vegetable image features and performs classification on unseen images. Further improvements can be made through hyperparameter tuning and dataset expansion.
